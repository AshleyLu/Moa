gap4export
------------------------------------------------

**Assembly export using gap4** - Export data from an assembly using gap4

Commands
~~~~~~~~

**clean**
  Remove all job data, not the Moa job itself, note that this must be implemented by the template.

**run**
  Export data from an assembly using gap4
