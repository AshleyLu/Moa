statsidx
------------------------------------------------




    Retrieve and print stats from BAM file to an index file



Commands
~~~~~~~~

**clean**
  Remove all job data, not the Moa job itself
  
  
**run**
  run samtools idxstats
  
  

Filesets
~~~~~~~~


**input**
  bam input files directory - forward files





**output**
  


  | *type*: `map`
  | *source*: `input`
  | *category*: `output`
  | *optional*: `{}`
  | *pattern*: `./*.index`
