Overview of Moa Commands

  • adhoc: Quickly create an adhoc analysis
  • cp: Copy a moa job (only the configuration, not the data), use moa cp DIR_FROM DIR_TO
  • help: Display help
  • history: display a version control log
  • intro: Show an introduction to Moa
  • kill: Kill a running moa job
  • list: List all known templates
  • listlong: List all known templates, showing a short description
  • manual_html: Generate a HTML manual of Moa
  • mv: Move a moa job,
  • new: Create a new Moa job in the current directory (unless -d is defined)
  • pack: pack a job or pipeline, or manage packs
  • pause: Pause a running moa job
  • rawinfo:
  • resume: Resume a paused moa job
  • set: Set, append, change or remove variables from the configuration of a Moa job.
  • show: Show the configured parameters and their values
  • status:
  • tag: Tag the current version
  • template: Show the template of this moa job
  • test: Run Moa unittests
  • tree: return a tree structure with extra moa information
  • unpack: unpack an earlier packed job/pipeline
