Moa template: lftp

lftp

DESCRIPTION

lftp

Targets

  • (empty) / lftp: Use LFTP to download files. This template has two modi, one is set lftp_mode to mirror data, in which case both lftp_url and lftp_pattern (default *) are used. The other modus is lftp_mode=get, when one file defined by lftp_url is downloaded. In the mirror mode it is possible to download only those files that are newer as the files already downloaded by using the lftp_timestamp parameter
  • all: Recursively run through all subdirectories (use make all action=XXX to run make XXX recursively)
  • clean: Remove all job data (not the job itself). This does works only if the template has properly implemented the clean target.

PARAMETERS

(* denotes a mandatory parameter)
  • *lftp_url: The base url to download from
    Data type: string.
  • lftp_dos2unix: Run dos2unix to prevent problems with possible dos text files
    Data type: set, T|F.
  • lftp_get_name: target name of the file to download
    Data type: string.
  • lftp_lock: Lock this job after running. This means that you will have to manually unlock the job before lftp actually reruns. This is a good choice if your downloading large datasets or have a slow connection
    Data type: set, T|F.
  • lftp_mode: Mode of operation - mirror or get. Mirror enables timestamping. Get just gets a single file. If using get, consider setting depend_lftp_timestamp to F. When using get, the full url should be in lftp_url. lftp_pattern is ignored. Defaults to mirror.
    Data type: set, mirror|get.
  • lftp_output_dir: subdir to create & write all output to. If not defined, data will be downloaded to directory containing the Makefile
    Data type: directory, Default: ".".
  • lftp_pass: password for the remote site, note that this can be defined on the commandline using: make lftp_pass=PASSWORD
    Data type: password.
  • lftp_pattern: glob pattern to download
    Data type: string, Default: "*".
  • lftp_powerclean: Do brute force cleaning (T/F). Remove all files, except moa.mk & Makefile when calling make clean. Defaults to F.
    Data type: set, T|F.
  • lftp_timestamp: Depend on lftp to decide if a file needs updating, else a touchfile is created that you need to delete or touch before updating (T/*F*)
    Data type: set, T|F.
  • lftp_user: username for the remote site
    Data type: string.

System parameters

  • *title: A name for this job
    Data type: string.
  • description: A longer description for this job
    Data type: string.

Advanced parameters

  • lftp_noclean: set of files not to be deleted by the powerclean
    Data type: string, Default: "moa.mk Makefile".
  • moa_postcommand: A single shell command to be executed after the Moa is finished. For more complex processing please override the moa_postprocess target in the local Makefile.
    Data type: string.
  • moa_precommand: A single command to be executed before the main operation starts. For more complicated processing, please override the moa_preprocess target in the local Makefile.
    Data type: string.
