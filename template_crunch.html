Moa template: crunch

Create ACT crunch files for use with Artemis ACT

DESCRIPTION

Create ACT crunch files for use with Artemis ACT

Targets

  • (empty) / crunch: Create a crunch file for use with the Artemis ACT comparison tool.
  • all: Recursively run through all subdirectories (use make all action=XXX to run make XXX recursively)
  • clean: Remove all job data, not the Moa job itself, note that this must be implemented by the template.

PARAMETERS

(* denotes a mandatory parameter)
  • crunch_eval: e value cutoff
    Data type: float, Default: "1e-10".
  • crunch_nothreads: threads to run crunch with (note the overlap with the Make -j parameter)
    Data type: integer, Default: "4".

Input parameters

  • *crunch_input_dir: Directory with input fasta files
    Data type: directory.
  • crunch_input_extension: file extension for the files in crunch_input_dir
    Data type: string, Default: "fasta".
  • crunch_input_glob: glob to select a subset of files from crunch_input_dir
    Data type: string, Default: "*".
  • crunch_input_limit: Number of files to use, if not defined: all files
    Data type: integer.
  • crunch_input_sort: Sort order. Choose from: u - unsorted, s - size, sr - size reverse, t - time, tr - time reverse
    Data type: set, u|s|sr|t|tr.

System parameters

  • *title: A name for this job
    Data type: string.

Advanced parameters

  • moa_postcommand: A single shell command to be executed after the Moa is finished. For more complex processing please override the moa_postprocess target in the local Makefile.
    Data type: string.
  • moa_precommand: A single command to be executed before the main operation starts. For more complicated processing, please override the moa_preprocess target in the local Makefile.
    Data type: string.
