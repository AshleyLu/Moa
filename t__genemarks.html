genemarks.mk

predict genes using geneMarkS

Targets

(empty)
Leaving the target unspecified executes the default target(s): (genemarks)
clean
removes all results from this job
all
executes the default target and into subdirectories to execute any other moa makefile it encounters

genemarks :

Parameters

Required parameters

genemarks_input_dir
directory containing the input sequences
genemarks_matrix
the matrix to use

Optional parameters

genemarks_gff_source
source field to use in the gff. Defaults to geneMarkS
genemarks_input_extension
input file extension. Defaults to 'fasta'
jid
Unique identifier for this job. Jids are autogenerated if undefined. A descriptive jid is important, particularly if you are using couchdb.
project
A project name - group your analyses.
