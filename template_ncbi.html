Moa template: ncbi

"Download from NCBI"

DESCRIPTION

"Download from NCBI"

Targets

  • (empty) / ncbi: Download a set of sequences from NCBI based on a query string (ncbi_query) and database (ncbi_db). This tempate will run only once (!), after a succesful run it creates a lock file that you need to remove to rerun
  • all: Recursively run through all subdirectories (use make all action=XXX to run make XXX recursively)
  • clean: Remove all job data (not the job itself). This does works only if the template has properly implemented the clean target.

PARAMETERS

(* denotes a mandatory parameter)
  • *ncbi_query: NCBI query (for example txid9397[Organism%3Aexp])
    Data type: string.
  • ncbi_db: NCBI database
    Data type: string, Default: "nuccore".
  • ncbi_sequence_name: Sequence name to download. When this parameter is set, the template assumes that only one sequence is to be downloaded, the rest will be discarded.
    Data type: string.

System parameters

  • *title: A name for this job
    Data type: string.
