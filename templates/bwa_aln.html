bwa_aln¶

Use BWA to align a set of fastq reads against a db

Commands¶

clean
Remove all job data, not the Moa job itself, note that this must be implemented by the template.
run
run bwa aln

Filesets¶

Fastq input files

output

category: output
optional: {}
pattern: ./*.sai
