gsMapper.mk

Targets

(empty)
Leaving the target unspecified executes the default target(s): (gsmap)
clean
removes all results from this job
all
executes the default target and into subdirectories to execute any other moa makefile it encounters

gsmap :

Parameters

Required parameters

gsmap_sfffile
SFF files with reads to map against the reference sequences (file)
gsmap_name
Name identifying this mapping in the output gff (filestring)
gsmap_reference_fasta
A multifasta file with the reference sequence(s)with the library id. (file)
title
A job name - Describe what you are doing (string)

Optional parameters

moa_precommand
A single command to be executed before the main operation starts. For more complicated processing, please override the moa_preprocess target in the local Makefile. (string)
moa_postcommand
A single shell command to be executed after the Moa is finished. For more complex processing please override the moa_postprocess target in the local Makefile. (string)
gsmap_min_overlap_len
Minimum overlap length in the assembly step (integer, default:40)
gsmap_min_overlap_ident
Minimum identity length in the assembly step (integer, default:90)
gsmap_annotation
Gene annotation file in the UCSC GenePred format (file)
