A function to start Make in a certain directory d with specific args

:param wd: Directory in which to execute make :type wd: String :param args: Arguments to pass to make :type args: String or List of Strings :param verbose: Have make be silent or generate lots of output :type verbose: Boolean :param captureOut: If True the output will be written to moa.out and moa.err :type captureOut: Boolean
