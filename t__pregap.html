pregap.mk

Targets

(empty)
Leaving the target unspecified executes the default target(s): (pregap)
clean
removes all results from this job
all
executes the default target and into subdirectories to execute any other moa makefile it encounters
pregap
Run pregap

Parameters

Required parameters

input_dir
Directory with the input data
input_pattern
file name pattern
cloning_vector
File containing the cloning vector
sequencing_vector
File containing the sequencing vector
ecoli_screenseq
File containing ecoli screen sequences
repeat_masker_lib
File with a repeatmasker library
vector_primerfile
File with the vector primers

Optional parameters

quality_value_clip
quality cutoff (default=10)
pregap_template
the template pregap config file to use. if not defined, Moa tries ./files/pregap.config.
jid
Unique identifier for this job. Jids are autogenerated if undefined. A descriptive jid is important, particularly if you are using couchdb.
project
A project name - group your analyses.
