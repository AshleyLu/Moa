Yaco¶

Yaco provides a dict like structure that can be serialized to & from yaml

Yaco can be found from the Python package index and is also part of the Moa source distribution

class Yaco.Yaco(data={})¶

Loosely based on http://code.activestate.com/recipes/473786/ (r1)

>>> v= Yaco()
>>> v.a = 1
>>> assert(v.a == 1)
>>> assert(v['a'] == 1)
>>> v= Yaco({'a':1})
>>> assert(v.a == 1)
>>> assert(v['a'] == 1)
