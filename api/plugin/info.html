moa.plugin.info.rawCommands(data)¶

(private) moa raw_commands - Print a list of all known commands

Usage:

moa raw_commands

Print a list of known Moa commands, both global, plugin defined commands as template specified ones. This command is mainly used by software interacting with Moa.

(private) moa raw_parameters - Print out a list of all known parameters

Usage:

moa raw_parameters
