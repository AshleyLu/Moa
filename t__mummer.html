mummer.mk

Targets

(empty)
Leaving the target unspecified executes the default target(s): (mummer)
clean
removes all results from this job
all
executes the default target and into subdirectories to execute any other moa makefile it encounters

mummer :

Parameters

Required parameters

mum_input_dir_a
This set is compared to the sequences in input_dir_b. only a forward comparison is made (a against b, not the other way round )
mum_input_dir_b
The set to compare against

Optional parameters

mum_input_extension
undefined
mum_breaklen
Set the distance an alignment extension will attempt to extend poor scoring regions before giving up (default 200)
jid
Unique identifier for this job. Jids are autogenerated if undefined. A descriptive jid is important, particularly if you are using couchdb.
project
A project name; group your analyses.
